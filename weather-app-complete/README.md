# Weather Monitoring 3-Tier Docker App (Complete)

This project contains a responsive Weather Monitoring application (Frontend - Backend - Database) that is Docker-ready with best practices applied.

## Run locally
1. Copy env:
   ```
   cp .env.example .env
   ```
   Edit `.env` and add your OpenWeather API key.

2. Build and run:
   ```
   docker compose up --build
   ```

3. Open UI:
   http://localhost:3000
